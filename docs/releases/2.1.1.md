# AriaNg 2.1.1

本次补丁版修复任务刷新、BitTorrent 小文件过滤与发布流程的可靠性问题。

## 修复

- WebSocket 关闭且未启用自动重连时，未完成 RPC 会明确失败，后续请求不会卡入已关闭的连接队列。
- 任务列表文件明细请求丢失时，30 秒后自动恢复刷新，不再永久阻塞。
- “稍后下载”的 BT 任务在恢复时若 child 已进入 active/waiting，会先暂停再筛选，始终保持“稍后下载”的暂停语义。
- 已经 active/waiting 的“立即下载” BT child 筛选完成后不再发送无效的 `unpause`。
- 第三次 `changeOption` 回包不确定时先对账已生效的选择，避免不必要地恢复全选。
- Release 手动输入版本不再直接插入 Bash，消除脚本注入风险。

## 文档

- 新增任务列表内嵌文件列表与 BT 小文件过滤的维护 PRD，明确当前行为、边界与维护决策原则。
