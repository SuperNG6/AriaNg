# AriaNg 2.2.0

本版本集中完善本 fork 新增的任务文件视图和 BT 小文件过滤功能。

## 新功能

- 在任务列表中提供可记忆的内嵌文件视图，并保持展开内容为只读展示。
- 新增对正在下载 BT 任务的批量小文件过滤，执行前先分析范围并二次确认。
- 为待过滤任务展示与实际处理阶段一致的状态提示。

## 稳定性与一致性

- 加强磁力任务父子 GID 恢复、活动子任务发现、断线重启和轮询生命周期处理。
- 保留手动文件排除，并在过滤失败时尝试恢复原选择。
- 统一自动过滤与批量过滤的排队优先级、并发行为和完成摘要。
- 修正任务菜单、文件视图和过滤服务之间的状态一致性。

## 界面与发布

- 精简批量过滤菜单文案，将风险与处理范围集中到确认弹窗并按行展示。
- 完成 11 种语言的 BT 过滤文案同步和占位符校验。
- 增加窄屏明暗主题、标准构建和 All-In-One 构建的发版验证。
